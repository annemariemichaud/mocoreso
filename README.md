# mocoreso

## Objectifs : 
Mocoreso était une application android utilisable par les élèves de 6ème en SVT dans la partie "Découverte de l'environnement proche".
Elle permettait d'enregistrer des observations après les avoir photographiées et de les localiser sur une google maps.

## Etat d'avancement du projet
Le projet n'est plus actif. L'application n'est plus disponible sur le google play store.
